Questions:
- Alerts dialogs stop other scripts. What are best practices for communicating errors or success messages to the user?
Notes:
- window object acts as a place to store global variables
- window.alert(), window.confirm(), and window.prompt() pauses any other scripts and waits for user input
- data attribute a way to add private data to an element for the sole purpose of being accessed by that pages JavaScript
- geolocation - where is the user?
- worker class - a way to sort of let the script run seperate threads. good for user experience so a page won't "hang"
- Multimedia - video and audio tags for embedding media to the page
- check the meta data is loaded before attempting to change values
video.addEventListener('loadedmetadata', () => { console.log(video.duration); });
