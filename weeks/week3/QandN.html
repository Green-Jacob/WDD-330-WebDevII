Questions:
- JavaScipt not being an object oriented language, how common are objects?
Notes:

Objects

-Objects in JS seem to be key pair arrays.
-Keyword this is for referring to members in the same object. Just like in C++ for example.
-Namespacing similar to C++. Functions defined inside objects must be called using the objects name. myObject.myFunction()

Document Object Model (DOM)

-DOM essential creates an object from the HTML file the JS is used on.
-Parts of the document, nodes, can be referenced like members of objects.
-All attributes of the html elements can be accessed and manipulated.
-CSS/style of the elements can be changed by the DOM.

Events

-Tie users actions to specific functions.
